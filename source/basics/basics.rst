##############
Basic Elements
##############

The ``basics`` folder contains the most basic elements of a documentation. It contains the most basic elements of a documentation, such as headings, paragraphs, lists, code blocks, etc.

The 'part' heading is created with::

   ##############
   Basic Elements
   ##############

In this document, **this** ``.rst`` file represents the 'Part' level of the documentation, and the 'Sectioning' document (``sectioning/sectioning.rst``) represents the 'Chapter' level. The 'Sectioning' document contains examples of all the different sectioning levels, and how they look like when rendered.
