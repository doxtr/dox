***************
Text Formatting
***************

The basic text formatting options in reStructuredText include:

Emphasis
   You can use single asterisks (*) to indicate emphasis, which is typically rendered in italics. For example, *this text is emphasized* will be rendered as this text is emphasized.

Strong
   You can use double asterisks (**) to indicate strong emphasis, which is typically rendered in bold. For example, **this text is strongly emphasized** will be rendered as this text is strongly emphasized.

Inline Code
   You can use backticks (\`\`) to indicate inline code. For example, ````this text is code```` will be rendered as ``this text is code``.

Literal Blocks
   You can use double colons (::) followed by an indented block of text to indicate a literal block, which is typically rendered in a monospaced font.

   For example::

      This is a literal block of text.
      It is typically rendered in a monospaced font.
